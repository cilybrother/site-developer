=========================================================
易度开放 API
=========================================================

易度是一个开放的平台，提供全套的操作API。具体包括：


- WebDAV API: 用于文档创建、删除、修改、查询等
- 人员和组织结构API：用于人员和组织结构创建、删除、同步等
- 流程API：用于流程的集成
- CAS SSO API：用于CAS的单点登录
- 简单单点登录API：用于简化的单点登录

易度webdav接口
----------------------------
易度提供webdav的接口，用于访问易度文档管理系统中保存的文件。
