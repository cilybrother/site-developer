=========================

这是一本易度PaaS开发的教程文档，本文将在http://everydo.com上发布。

易度办公平台上，利用PaaS开发平台，可直接在浏览器上快速开发企业应用。

详细内容，请查看 contents/edo-paas/api

如何运行这个站点
===========================
本站点使用ZCMS构建，运行方法为：

   python bootstrap.py
   bin/buildout
   bin/pserve development.ini

详细参见：

  https://github.com/everydo/zcms
